# FaceMesh - Real-Time Facial Attention Tracker

A browser-based real-time facial attention tracking system built using TensorFlow.js FaceMesh and React.js. The application analyzes facial landmarks to monitor user attention, detect drowsiness, track blinks, estimate head direction, and identify smiles in real time.

## Features

- 🎯 Real-time facial landmark detection using TensorFlow.js FaceMesh
- 👁️ Blink detection using Eye Aspect Ratio (EAR)
- 😴 Drowsiness detection based on continuous eye closure
- 😊 Smile detection using facial landmark analysis
- 🧭 Face direction and head pose estimation
- 📈 Live EAR analytics dashboard
- ⚡ Browser-based real-time inference with no backend required
- 🌐 Deployed on Vercel

## Tech Stack

- React.js
- TensorFlow.js
- FaceMesh Model
- JavaScript
- HTML5
- CSS3
- Vercel

## Key Highlights

- Built a real-time facial attention monitoring system using TensorFlow.js FaceMesh.
- Processed all 468 facial landmarks to perform facial behavior analysis directly in the browser.
- Implemented blink detection and drowsiness monitoring using Eye Aspect Ratio (EAR) calculations.
- Developed face direction estimation and smile detection using facial landmark geometry.
- Designed a live analytics dashboard for visualizing EAR values and user attention metrics.
- Optimized the application for low-latency, browser-based inference and deployed it on Vercel.
